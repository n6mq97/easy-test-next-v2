# Admin Module

Этот модуль отвечает за функционал, доступный только пользователям с ролью администратора.

## Frontend (components)

- `AdminPanel`: UI-компонент с элементами управления для административных задач (например, экспорт/импорт БД).

## Backend (lib)

- `actions.ts`: Содержит Server Actions для выполнения административных задач.
